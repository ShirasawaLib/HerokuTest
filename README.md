# HerokuTest

こちらのソースはHerokuでnodejsをデプロイして動かすためのサンプルコードです。  
必要があれば適宜クローンしてご利用ください。  

## Herokuデプロイ手順
1. heroku 登録  
1. eroku CLIダウンロード+インストール  
1. heroku login  
1. heroku create  
1. git push heroku master  
1. heroku create時に生成されるURLにアクセス
